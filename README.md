# SuperFU (Super Follow-UP) 🚀
**The AI-Powered Networking Assistant: Turning handshakes into high-value relationships.**

SuperFU is an intelligent application built to solve the "lost lead" problem. By using **Gemini AI**, it automates the process of capturing contact data from business cards and drafting high-context follow-up messages instantly.

---

## ✨ Core Features

* **📸 AI Business Card Scanning:** Uses Gemini Vision to extract names, emails, and company details from photos—eliminating manual data entry.
* **✍️ Intelligent Contact Profiles:** Creates rich, context-aware profiles for every person you meet, including interaction history.
* **📧 AI-Powered Follow-ups:** Automatically generates tailored follow-up drafts (Email or LinkedIn) based on the specific context of your meeting.
* **🎙️ Voice-to-Text Notes:** A built-in transcriber to dictate notes immediately after a meeting, ensuring no detail is forgotten.
* **🤖 Networking Chatbot:** An integrated AI assistant to suggest networking strategies or query your contact database.

---

## 🛠️ The Tech Stack
* **LLM Engine:** Google Gemini Pro & 1.5 Flash (via Google AI Studio)
* **Framework:** Node.js / React
* **Integrations:** Gmail API & Google ADK (Multi-Agent logic)

## 💻 How to Run & Deploy Locally
*This project was built and exported via Google AI Studio.*

### Prerequisites
* **Node.js** (v18 or higher recommended)

### Setup
1. **Clone the repository:**
   `git clone [Your-Repo-URL]`
2. **Install dependencies:**
   `npm install`
3. **Configure API Key:**
   Set your `GEMINI_API_KEY` in the `.env.local` file.
4. **Run the app:**
   `npm run dev`
